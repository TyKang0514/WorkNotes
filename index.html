Projects

Can We Predict the Winner of a Match in World Cup?

Data: World Cup Match Results (1930 - 2022)
Tools Used: Python, Tableau

There is always one of our friends saying, 'Let's bet on the match!'. This work is to identify some factors that help predicting the outcome of World Cup matches, based on features from every single match in history.

Who Is Not Paying?

Data: Mock data (customer info and financial transaction records)
Tools Used: SQL, Tableau

This project imagines a real-life business situation and builds a dashboard to keep track of who hasn’t paid yet. It helps spot which customers still owe money, so the business can follow up and manage cash flow better.

Who Is More Likely to Default on Their Loan?

Data: Loan Application
Tools Used: Python (Pandas, Scipy, Matplotlib)

Using a loan defaulter dataset, this project identifies groups with common features that have higher or lower default rates than the overall population. The Z-test, a common statistical method for inference, is applied to determine which default rates can be considered significantly high or low.

Are Coastal Homes Really More Expensive?

Data: 1990 California Housing Prices
Tools Used: Python (Pandas, Scipy, Matplotlib)

If we want to understand something about a group — like housing prices in a certain area — we usually rely on a sample. That’s where inferential statistics come in, and the t-test is one commonly used method. In this project, t-tests are used to examine housing prices in specific areas, based on the 1990 California housing dataset.
